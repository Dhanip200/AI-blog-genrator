🧠 BlogAgenticAI
A simple Flask web app that lets you generate blog summaries, AI thoughts, audio narration, and chat with an AI agent—all based on your input topic and language.

🚀 Features
🔍 Generate blog summaries based on any topic

💡 Get AI-generated thoughts for deeper insight

🔊 Audio narration of the summary (TTS-ready)

💬 Real-time AI chat agent for follow-up questions

🌐 Multi-language support

⚙️ Built on a modular agent framework
